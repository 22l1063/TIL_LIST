# hover_style

## 学んだこと
* 疑似クラス．疑似要素について
* `hover`疑似クラスについて

### 疑似クラスとは
要素がある特定の状態になったときだけ反応するスタイル。（例.テキストにカーソルが乗ったときに色が変わる。）
`hover`は疑似クラスである。

### HTML
```
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Station 7</title>
    <link rel="stylesheet" href="./assets/css/station7.css" type="text/css" />
  </head>
  <body>
    <p><a href="https://techtrain.dev/mypage">TechTrain マイページリンク</a></p>
  </body>
</html>
```
### CSS
```
a:hover {
    color: #008000;
}
```
疑似クラスは　`:疑似クラス`とかく。
`:hover`をつけることで、TechTrain マイページリンクにカーソルが乗るとCSSで指定した色になる。

### 疑似要素とは
コードを省略できる。HTMLがぐちゃぐちゃになるのを防ぐ。
今回は代表的な`::after`と`::before`を紹介する。

具体的には

### HTML
```
<p class="price">
  100
  <span>円</span>
</p>
<p class="price">
  500
  <span>円</span>
</p>
<p class="price">
  1000
  <span>円</span>
</p>
```

### CSS
```
.price{
  font-size: 15px;
}

.price span{
  font-size: 13px;
}
```

このコードは何回も`<span>`と円を書かなくてはいけない。
これを疑似要素を使って書くとこうなる。

### HTML
```
<p class="price">100</p>
<p class="price">500</p>
<p class="price">1000</p>
```

### CSS
```
.price{
  font-size: 15px;
}

.price::after{
  content: "円";
  font-size: 13px;
}

.price::before{
  content: "これは、";
  font-size: 13px;
}
```
priceクラスの`::before`,`::after`つまり前と後ろに`content:`の内容を表示するというコードである。
